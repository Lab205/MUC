Aalta目录结构描述
=================

主要目录结构
--------------------
```
.
├── Makefile                     ─── 项目构建配置文件
├── README.md                    ─── 项目的总体说明文件
├── build                        ─── 项目编译目录
├── dist                         ─── 分发目录
├── docs                         ─── 保存项目各种文档
├── lib                          ─── 外部依赖库目录
├── main.cpp                     ─── 程序入口
├── nbproject                    ─── netbeans配置文件
├── res                          ─── 资源目录
├── src                          ─── 源代码
├── tests                        ─── 测试
└── tools                        ─── 项目支撑工具目录
```
